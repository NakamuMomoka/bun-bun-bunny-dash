# Bun Bun Bunny Dash MVP 仕様

## コンセプト

2Dドット絵美少女が横方向に走りながら、自動攻撃で敵を倒し、レベルアップ3択で強化していく小規模アクション。

## コア体験

- 左右移動のみで直感的に操作できる
- プレイヤーは自動で攻撃する
- 敵を倒して経験値を集める
- レベルアップ時に3択の強化を選ぶ
- なるべく長く生き残る

## シーン構成

- TitleScene
- GameScene
- ResultScene

## GameScene の必須要素

- Player
- EnemySpawner
- Canvas
  - HP 表示
  - Level 表示
  - EXP バー
  - LevelUpPanel
- Background
- GameManager

## 最初の強化候補

- 発射間隔短縮
- 弾ダメージ上昇
- 最大HP上昇

## MVP 完了条件

- タイトルからゲーム開始できる
- プレイヤーが左右移動できる
- 一定間隔で自動攻撃が出る
- 敵がスポーンして移動する
- 弾と敵の当たり判定がある
- 被弾でHPが減る
- 敵撃破で経験値が増える
- レベルアップ時に3択UIが出る
- HP0でリザルトへ遷移する
